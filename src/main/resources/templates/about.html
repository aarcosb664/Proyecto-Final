About this Project

Breakout! - Final DAW Project

This project is a modern web version of the classic Breakout game, created as the final project for the DAW cycle by Alejandro Arcos Bellido.
Enjoy the game, compete in the ranking, and discover other games from the community!

  • Classic arcade gameplay with a modern twist
  • Power-ups, levels, and global ranking
  • Community-driven content

Technologies Used

Bootstrap 5 JavaScript HTML5 CSS3 Spring Boot Thymeleaf MySQL GitHub Cloudinary
